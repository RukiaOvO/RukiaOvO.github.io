# RukiaOvO.github.io

基于 **Hugo + Blowfish + GitHub Pages** 的个人主页。

这是一个单人维护的个人站点，用于展示个人简介、项目与文章内容。

## 站点定位

当前主页包含：

- 首页：快速介绍与导航入口
- 项目：展示个人项目与实践
- 文章：记录学习与技术笔记
- 关于：介绍个人背景与关注方向

## 技术栈

- [Hugo Extended](https://gohugo.io/)
- [Blowfish](https://blowfish.page/)
- [GitHub Pages](https://pages.github.com/)
- Git submodule 管理主题
